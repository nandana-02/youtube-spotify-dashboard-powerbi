# YouTube & Spotify Dashboard (Power BI)

## Project Overview

This project presents an interactive dashboard built using Power BI to analyze user engagement and content performance across YouTube and Spotify platforms.

## Tools Used

* Power BI
* Data Cleaning & Transformation
* CSV / Excel Dataset

## Key Metrics

* Views, Likes, Subscribers (YouTube)
* Streams, Followers (Spotify)

## Features

* Interactive dashboard with filters
* Platform-wise comparison
* Trend analysis of engagement metrics
* Dynamic visualizations for better insights

## Insights

* Identified high-performing content trends
* Compared engagement across platforms
* Analyzed growth patterns over time

## How to Use

Download the .pbix file and open it using Power BI Desktop to explore the dashboard.
